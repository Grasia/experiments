Monogatari: Visual Novel Engine

Built for the modern web, making it easy to create and distribute visual novels that anyone can play pretty much anywhere! You don't need to be a developer to start creating a game, Monogatari features a friendly language designed to be incredibly simple and yet very powerful, which will allow you to write your story as easily as if you were just writing it. Want to show images or videos? Maybe even play some music and sound effects! With Monogatari, you can use all sorts of media to enhance the experience of your game and provide richer content and experience. Monogatari comes pre-loaded with features you and your players will love such as load and save support, animations, ability to go back to previous states, autoplay mode, ability to skip text, and more! With Monogatari, you are in control of your visual novel like never before! You can change, add and remove anything you need. Think of your game as a website, everything you've seen before on any website is now within your reach! Monogatari is an open-source project released under the MIT License which means you can use it for any commercial or non-commercial project without any limitation and it's completely free! Not only can you release your game to the web where anyone will be able to play it from their browser, but you can also build your game for Windows, macOS, Linux, Android, and iOS!

Prerrequisites:

Monogatari has been created specifically to be a web page, even before a game. Therefore, to develop in Monogatari you need the same as to develop a web page. No installation is required, unless you want more advanced features as we will explain later. You will need a text editor capable of editing HTML, Javascript, and CSS. Any text editor should be able to do this, but we recommend one with code syntax highlighting. Some recommended (and free) include:

  • Visual Studio Code
  • Atom
  • Brackets

Now, you can always open your graphic novel by clicking on the index.html file and opening it with your browser, however there are little aspects of Monogatari that work best when served through a web server. You don't need anything fancy for this, in fact there is a perfectly good web server that you can download from the Chrome Store. As mentioned above, using a web server is completely optional, you can open your game with the browser as a file and it will work. OK, the web server will allow you to test features like service workers, required for Monogatari offline support and asset preloading.
